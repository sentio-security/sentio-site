# sentio-site

Landing page for [sentio](https://github.com/sentio-security/sentio-rs) — a security scanner for Solana programs.

Live at [sentiosecurity.xyz](https://sentiosecurity.xyz)

## Stack

- [Next.js 16](https://nextjs.org) — App Router, static export
- [Tailwind CSS v4](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [DM Mono](https://fonts.google.com/specimen/DM+Mono) via next/font

## Dev

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).
